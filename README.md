# rzpaas_sdk_demo_264_apple

展示RZPaasSDK如何使用外部H264的自定义数据源

# sdk [下载地址](https://docs.rzrtc.com/docs/paas/download/)

## 运行demo

1. 替换 AppId , 在文件 `KeyCenter.swift`

    ```
    import Foundation
    //MARK: - replace me
    public let kAppId = ""
    ```

2. 执行`pod install`
3. 运行demo, ios/mac
    - `rzpaas_sdk_demo_264_ios`
    - `rzpaas_sdk_demo_264_macos`
